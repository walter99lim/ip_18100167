1. 다음 문장에 ○ 또는 ×를 표시하시오.
(1) 웹은 1990년대 중반부터 모든 대중이 사용할 수 있게 되었습니다. (○)
1990년대 중반, 팀 버너스리가 월드 와이드 웹(WWW)을 개발하여 일반 대중도 웹을 사용할 수 있게 되었다.
(2) 모든 URI는 URL입니다. (×)
URI(Uniform Resource Identifier)는 URL(Uniform Resource Locator)과 URN(Uniform Resource Name)을 포함하는 개념이다. 모든 URI가 URL은 아닙니다.
(3) 2000년대 초에는 ActiveX와 플래시 덕분에 웹 브라우저로 다양한 것을 할 수 있었습니다. (○)
2000년대 초반 ActiveX와 플래시는 웹에서 멀티미디어 콘텐츠를 구현하는 데 많이 사용되었다.
(4) 자바스크립트를 사용하면 웹 페이지에 프로그래밍 요소를 부여해서 사용자와 반응을 할 수 있습니다. (○)
자바스크립트는 웹 페이지에 동적 동작과 상호작용을 추가할 수 있다.
(5) HTML5가 등장하며 그레디언트, 애니메이션 등 화려한 스타일을 CSS3로 추가할 수 있게 되었습니다. (○)
HTML5와 함께 등장한 CSS3는 그레디언트, 애니메이션, 변환(transform) 등을 지원한다.
(6) HTML5는 단방향 통신뿐만 아니라 양방향 통신도 가능합니다. (○)
HTML5는 WebSocket을 통해 실시간 양방향 통신을 지원한다.
(7) HTML5는 배터리 잔량 같은 장치 정보를 가져올 수 있습니다. (○)
HTML5는 Geolocation API, Battery API 등을 통해 디바이스 정보를 가져올 수 있다.
(8) 웹 브라우저에서 작동하는 클라이언트 프로그램은 기본적으로 C++, 자바, C#, 파이썬 등의 프로그램으로 개발됩니다. (×)
웹 브라우저에서 작동하는 클라이언트 프로그램은 주로 HTML, CSS, JavaScript로 작성된다.
(9) 인터넷의 모태는 ARPANET입니다. (○)
ARPANET은 인터넷의 초기 형태로, 미국 국방부에서 개발되었다.

02. HTML5를 개발하고 표준화한 주요 조직은 무엇인가?
정답: ① W3C, WHATWG
W3C(World Wide Web Consortium)와 WHATWG(Web Hypertext Application Technology Working Group)는 HTML5의 개발과 표준화를 담당한 주요 조직이다.

03. HTML5가 도입되면서 더 이상 사용하지 않게 된 기술을 모두 고르시오.
정답: ② Flash, ④ ActiveX
HTML5는 멀티미디어 콘텐츠를 네이티브 방식으로 처리할 수 있어 Flash와 ActiveX의 필요성이 줄어들었다.

04. 다음 내용을 설명하시오.
(1) HTML5를 큰 의미와 작은 의미로 나누어 설명하시오.
큰 의미: HTML5는 HTML, CSS3, JavaScript를 포함한 웹 표준 기술을 통합적으로 지칭한다.
작은 의미: HTML5는 HTML 언어의 새로운 버전으로, 시맨틱 태그(<header>, <footer>, <article> 등)를 포함한다.
(2) HTML5를 사용해 스마트폰 애플리케이션을 개발할 때 장점을 설명하시오.
HTML5는 멀티플랫폼 지원이 가능하며, 웹 앱으로 개발 시 여러 운영 체제에서 동작한다.
추가로 설치 없이 브라우저만으로 실행 가능하며, 유지보수가 용이하다.

05. 개발자들이 "Awesome O O"와 관련된 것을 정리한 Awesome HTML 페이지를 확인하시오.
정답: Awesome HTML 페이지는 다양한 HTML5의 활용 사례, 리소스, 라이브러리 등을 정리한 GitHub 문서다. HTML5 학습 및 실무 참고 자료로 활용된다.

06. 웹 2.0 사례를 찾아 발표하시오.
대표적인 예: 블로그, 위키, 소셜 미디어.
