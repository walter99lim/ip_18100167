1. 다음 문장에 ○ 또는 ×를 표시하시오.
(1) HTML 태그는 반드시 내부에 무언가를 포함해야 합니다. (×)
HTML 태그는 반드시 내부에 콘텐츠를 포함하지 않아도 된다. 예: <br>, <img> 같은 빈 태그가 존재한다.
(2) HTML 태그 내부에는 텍스트만 입력할 수 있습니다. (×)
HTML 태그 내부에는 텍스트뿐 아니라 다른 HTML 태그, 속성 등이 올 수 있다.
(3) HTML 태그의 lang 속성은 구글 등 검색 엔진에 문서의 언어 정보를 알려줄 때 사용됩니다. (○)
lang 속성은 검색 엔진과 브라우저가 문서의 언어를 파악하도록 도와준다.
